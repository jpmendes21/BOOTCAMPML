1 - Explique, com suas palavras, o que é machine learning?

É a parte da Inteligência Artificial que permite que os computadores aprendam utilizando dados a identificar padrões, fazer previsões sem serem programados antes. É feito treinamentos para que ela seja ainda mais precisa e melhore seu desempenho a medida que é alimentada de informações.

2 - Explique o conceito de conjunto de treinamento, conjunto de validação e 
conjunto de teste em machine learning.

Conjunto de treinamento: É onde o modelo vê exemplos e aprende padrões a fim de melhorar sua precisão.

Conjunto de Validação: Funciona como um teste para que o desenvolvedor verifique o desempenho do modelo ainda durante a fase de criação.

Conjunto de teste: É a etapa final onde são introduzidos dados que o modelo não reconhece em nenhuma etapa anterior. Com objetivo de testar e aprovar a usabilidade do modelo.

3 - Explique como você lidaria com dados ausentes em um conjunto de dados 
de treinamento.

Caso seja uma quantidade pequena de valores, é possível excluir linhas, caso não é possível preencher com a normalização dos dados, baseado em outros dentro do modelo, ou utilizar outros para prever quais seriam os dados faltantes.

4 - O que é uma matriz de confusão e como ela é usada para avaliar o 
desempenho de um modelo preditivo?

A matriz de confusão é uma ferramenta usada para avaliar o desempenho de modelos de classificação em Machine Learning. Ela mostra quantas previsões o modelo acertou e quantas errou, separando os tipos de acertos e erros. Assim sendo possível avaliar a precisão do modelo durante o uso.
